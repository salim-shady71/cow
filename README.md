# Shreya's Analytics Quest Website

## Files
- `index.html` → the website
- `styles.css` → styling
- `app.js` → game logic
- `content.js` → the file you edit daily

## Daily update workflow
Open `content.js` and edit the `CURRENT_DAY` object.

Fields you will usually change:
- `day`
- `level`
- `title`
- `subtitle`
- `objective`
- `tools`
- `datasetName`
- `datasetUrl`
- `steps`
- `expectedOutput`
- `hint`
- `professorTip`
- `reflection`
- `xpReward`
- `message`
- `checkpoint`
- `reward`

## Run locally
Just open `index.html` in your browser.

## GitHub Pages hosting
1. Create a new GitHub repository.
2. Upload all files.
3. Go to Settings → Pages.
4. Choose:
   - Source: Deploy from branch
   - Branch: main / root
5. Save.

Your site will appear at:
`https://yourusername.github.io/repository-name/`

## Note
Progress is saved in the browser using localStorage.
